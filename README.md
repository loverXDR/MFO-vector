# Заначка — Умное финансирование (MFO-vector)

Это репозиторий статического сайта микрофинансовой организации "Заначка", обновленного до версии V2.
Все исходные файлы сайта (HTML, CSS, JS) находятся в папке `src/`.

## Структура проекта
- `src/` — исходные файлы сайта (index.html, styles.css, app.js и другие)
- `Dockerfile` — базовый образ Nginx
- `docker-compose.yml` — конфигурация для запуска контейнера с пробросом папки `src/` (bind mount)

## Запуск проекта

### Вариант 1: Локально (через Python)
Если у вас установлен Python, вы можете запустить простой HTTP-сервер перейдя в папку `src`:
```bash
cd src && python3 -m http.server 8000
```
Сайт будет доступен по адресу: http://localhost:8000

### Вариант 2: Через Docker (рекомендуется)
Контейнер использует Nginx. Файлы сайта "зашиты" непосредственно внутрь Docker-образа при сборке. Это значит, что контейнер полностью автономен и готов к "боевому" развертыванию (production-ready).

**Используя Docker Compose:**
1. Убедитесь, что у вас установлены Docker и Docker Compose.
2. Выполните команду в корне проекта:
   ```bash
   docker compose up --build -d
   ```
   *(Флаг `--build` нужен, если вы изменили файлы в папке `src/` и хотите пересобрать образ с новыми изменениями).*
3. Сайт будет доступен по адресу: http://localhost:8080

Чтобы остановить работу, введите: `docker compose down`

**Используя только Docker:**
```bash
docker build -t mfo-zanachka .
docker run -d -p 8080:80 --name zanachka-web mfo-zanachka
```
